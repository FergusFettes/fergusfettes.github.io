IsItPossible

Published

January 27, 2023

“Now, is it possible that some Israeli soldiers go berserk under pressure and wind up shooting into crowds of rock-throwing children? Of course.”

In this essay I am going to seek support for the following claim– that when the question ‘Is It Possible’ is used in rhetoric in this manner, it is always making a claim about permissibility.

The basic idea is that tautology often uderwrites metaphor– here stating that something actual is possible is the tautology, and moral permissibility is the metaphier.

Methodology

First, use the google books api to search for ‘is it possible’:

import requests


def get_books(query):
    url = "https://www.googleapis.com/books/v1/volumes"
    params = {"q": query, "maxResults": 40}
    r = requests.get(url, params=params)
    return r.json()


def get_books_by_author(author):
    return get_books("inauthor:" + author)


def get_books_by_title(title):
    return get_books("intitle:" + title)


def get_books_by_quote(quote):
    return get_books("intext:" + quote)


# Get books with the phrase "is it possible"
books = get_books_by_quote("is it possible")
